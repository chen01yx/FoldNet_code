# Codebase of FoldNet
This repository is the official implementation of [FoldNet](https://pku-epic.github.io/FoldNet/).

## install
```
git clone https://github.com/chen01yx/FoldNet_code.git --recurse-submodules
cd FoldNet_code
git submodule update --init --recursive
```
